## 概述

HelloWorld-Python 采用极简主义架构设计，将复杂度降到最低。

## 核心模块

### 字符串拼接引擎

负责将两个字符串字面量 `'Hello, '` 和 `"World!"` 通过 `+` 运算符合并。

- **输入**: `str` + `str`
- **输出**: 单个 `str`
- **时间复杂度**: O(n)
- **空间复杂度**: O(n)

### print() 输出系统

负责将拼接后的字符串写入标准输出流。

- **调用**: `print(content)`
- **目标**: `sys.stdout`
- **换行**: 自动追加 `\n`

## 数据流

```
字面量 'Hello, ' ──┐
                   ├──▶ 拼接 ──▶ 'Hello, World!' ──▶ print() ──▶ stdout
字面量 "World!"  ──┘
```

## 设计决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 字符串引号 | 混用单引号和双引号 | 展示 Python 语法灵活性 |
| 输出方式 | `print()` | 最直观的输出手段 |
| 依赖管理 | 零依赖 | 最大化可移植性 |
| 代码行数 | 2 行 | KISS 原则 |

## 扩展性

当前架构支持以下扩展方向：

1. **多语言版本** — 替换字符串字面量
2. **函数化封装** — 将逻辑提取为 `def main():`
3. **参数化输出** — 引入 `sys.argv` 接收命令行参数
4. **日志系统** — 将 `print()` 替换为 `logging` 模块
