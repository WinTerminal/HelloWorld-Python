## 概述

HelloWorld-Python 採用極簡主義架構設計，將複雜度降到最低。

## 系統架構圖

```
┌─────────────────────────────────────────┐
│              使用者終端機               │
│  ┌───────────────────────────────────┐  │

│  └───────────────┬───────────────────┘  │
│                  │                       │
│  ┌───────────────▼───────────────────┐  │
│  │          Python 直譯器            │  │
│  │  ┌─────────────────────────────┐  │  │

```

## 核心模組

### 字串串接引擎

負責將兩個字串字面值 `'Hello, '` 和 `"World!"` 透過 `+` 運算元合併。

- **輸入**: `str` + `str`
- **輸出**: 單個 `str`
- **時間複雜度**: O(n)
- **空間複雜度**: O(n)

### print() 輸出系統

負責將串接後的字串寫入標準輸出串流。

- **呼叫**: `print(content)`
- **目標**: `sys.stdout`
- **換行**: 自動追加 `\n`

## 資料流

```
字面值 'Hello, ' ──┐
                    ├──▶ 串接 ──▶ 'Hello, World!' ──▶ print() ──▶ stdout
字面值 "World!"  ──┘
```

## 設計決策

| 決策 | 選擇 | 理由 |
|------|------|------|
| 字串引號 | 混用單引號和雙引號 | 展示 Python 語法靈活度 |
| 輸出方式 | `print()` | 最直觀的輸出手段 |
| 依賴管理 | 零依賴 | 最大化可攜性 |
| 程式碼行數 | 2 行 | KISS 原則 |

## 擴展性

目前架構支援以下擴展方向：

1. **多語言版本** — 替換字串字面值
2. **函式化封裝** — 將邏輯提取為 `def main():`
3. **參數化輸出** — 引入 `sys.argv` 接收命令列參數
4. **日誌系統** — 將 `print()` 替換為 `logging` 模組
